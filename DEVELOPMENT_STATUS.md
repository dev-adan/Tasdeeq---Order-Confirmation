## 🚧 Next Steps (To Be Implemented)

### Phase 1: Order Webhook Handler
**File**: `app/routes/webhooks.orders.create.jsx`

**What it does**:
1. Receives order from Shopify
2. Checks subscription status and limits
3. Filters by order type (COD/prepaid)
4. Saves order to database
5. Sends WhatsApp confirmation message

**Key Logic**:
```javascript
// Check if shop has confirmations left
const canProcess = canProcessOrder(shop);
if (!canProcess.allowed) {
  // Send notification to merchant
  return;
}

// Check order type filter
const isCOD = payload.payment_gateway_names[0].includes("COD");
if (shop.orderTypeFilter === "cod" && !isCOD) return;

// Send WhatsApp message
await sendWhatsAppMessage(...);

// Increment counters
await db.shop.update({
  data: {
    confirmationsUsedThisMonth: shop.confirmationsUsedThisMonth + 1,
    confirmationsUsedToday: shop.confirmationsUsedToday + 1,
    confirmationsRemaining: shop.confirmationsRemaining - 1
  }
});
```

## 📊 Current User Flow

```
1. Merchant installs app
   ↓
2. Opens app → Dashboard (auto-creates free plan)
   ↓
3. Clicks "Upgrade Plan" → Plan selection page
   ↓
4. Chooses plan → Shopify billing
   ↓
5. Returns to app → Billing callback → Settings page
   ↓
6. Configures settings:
   - Order type filter
   - Courier settings
   ↓
7. App is active and ready to process orders
```
